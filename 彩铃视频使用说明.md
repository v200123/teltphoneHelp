# 彩铃视频使用说明

## 功能说明

应用支持在接通电话前播放彩铃视频，并支持**彩铃与手机号码绑定**功能：

- **一个彩铃可以绑定多个手机号码**
- **一个手机号码只能绑定一个彩铃**（或没有）
- 拨打电话时，根据被叫号码查找绑定的彩铃
- 如果号码没有绑定彩铃，则不播放彩铃（播放拨号音）

## 绑定管理界面

### 如何打开管理界面

在代码中启动绑定管理界面：

```kotlin
// 从 MainActivity 或其他地方启动
val intent = Intent(context, RingtoneBindingManageActivity::class.java)
startActivity(intent)
```

建议在「彩铃管理」界面或设置中添加入口：

```kotlin
// 在 RingVideoManageActivity 中添加按钮
binding.btnManageBinding.setOnClickListener {
    startActivity(Intent(this, RingtoneBindingManageActivity::class.java))
}
```

### 界面功能说明

打开 `RingtoneBindingManageActivity` 后：

1. **选择彩铃**：首次进入需要选择一个要管理的彩铃
2. **查看绑定**：显示当前彩铃已绑定的所有号码
3. **添加单个号码**：输入手机号点击添加
4. **批量导入**：一次导入多个号码（支持复制粘贴）
5. **解绑号码**：点击号码右侧的删除按钮
6. **清空全部**：一键解绑该彩铃的所有号码

### 批量导入格式

支持以下格式：
```
13800138000
13900139000
13500135000,13600136000
13700137000 13800138000
```

系统会自动识别所有有效的11位手机号码。

## 绑定关系说明

```
彩铃A ←───┬─── 号码13800138000
          ├─── 号码13900139000
          └─── 号码13700137000

彩铃B ←───┬─── 号码13600136000
          └─── 号码13500135000

号码13300133000 （未绑定任何彩铃）
```

## 播放优先级

```
号码绑定彩铃 > 随机彩铃 > res/raw > assets > 外部存储
```

### 详细逻辑

1. **首先检查号码是否绑定了彩铃**
   - 如果绑定了 → 播放绑定的彩铃
   - 如果没有绑定 → 进入下一步

2. **尝试随机播放数据库中的彩铃**（向后兼容）
   - 如果有彩铃 → 随机播放
   - 如果没有 → 进入下一步

3. **尝试默认视频位置**
   - res/raw → assets → 外部存储

4. **最终都没有找到**
   - 隐藏视频视图，播放拨号等待音

## 数据库表结构

### 彩铃表 (RingVideo)

```java
@Entity
public class RingVideo {
    @PrimaryKey(autoGenerate = true)
    public int id;
    
    public String videoName;    // 视频名称
    public String videoUri;     // 视频路径/URI
    public String mimeType;     // 视频类型
    public long fileSize;       // 文件大小
    public long duration;       // 视频时长
    public long createdAt;      // 创建时间
    public boolean isSelected;  // 是否为当前选中
}
```

### 号码绑定表 (RingtonePhoneBinding)

```java
@Entity(
    foreignKeys = @ForeignKey(
        entity = RingVideo.class,
        parentColumns = "id",
        childColumns = "ringtoneId",
        onDelete = ForeignKey.CASCADE // 彩铃删除时自动解绑
    ),
    indices = {
        @Index(value = {"phoneNumber"}, unique = true),
        @Index(value = {"ringtoneId"})
    }
)
public class RingtonePhoneBinding {
    @PrimaryKey(autoGenerate = true)
    public int id;
    
    public String phoneNumber;  // 手机号码（唯一）
    public int ringtoneId;      // 绑定的彩铃ID
    public long bindTime;       // 绑定时间
}
```

## DAO 操作方法

```java
// 绑定号码到彩铃
ringtonePhoneBindingDao().insert(binding);

// 重新绑定（先解绑再绑定）
ringtonePhoneBindingDao().rebindPhoneNumber(phoneNumber, ringtoneId);

// 查询号码绑定的彩铃ID
Integer ringtoneId = ringtonePhoneBindingDao().getRingtoneIdByPhone(phoneNumber);

// 解绑号码
ringtonePhoneBindingDao().deleteByPhoneNumber(phoneNumber);

// 获取彩铃绑定的所有号码
List<RingtonePhoneBinding> bindings = ringtonePhoneBindingDao().getByRingtoneId(ringtoneId);

// 检查号码是否已绑定
boolean isBound = ringtonePhoneBindingDao().isPhoneBound(phoneNumber) > 0;
```

## 视频播放调用

在通话界面中，系统会自动根据号码查找彩铃：

```kotlin
// 拨号时传入号码，自动查找绑定的彩铃
VideoPlayerHelper.getInstance().playRingtoneVideo(
    context, 
    packageName, 
    phoneNumber  // 电话号码
) { isVideoPlaying ->
    if (!isVideoPlaying) {
        // 没有视频时播放拨号等待音
        MediaPlayerHelper.getInstance().playCallSound(this)
    }
}
```

## 界面操作指南

### 添加单个号码绑定

1. 打开「彩铃号码绑定」界面
2. 选择一个彩铃（首次进入自动弹出选择）
3. 在输入框输入手机号
4. 点击「添加」按钮
5. 成功后会显示在列表中

### 批量导入号码

1. 点击右上角「批量导入」
2. 在文本框中粘贴号码（每行一个或用逗号/空格分隔）
3. 实时显示识别到的号码数量
4. 点击「导入」按钮
5. 成功后会添加到列表

### 切换管理的彩铃

1. 点击顶部当前彩铃名称区域
2. 弹出选择对话框
3. 选择要管理的彩铃
4. 界面会刷新显示该彩铃的绑定号码

### 解绑号码

1. 在列表中找到要解绑的号码
2. 点击右侧删除图标
3. 确认解绑

## 视频格式建议

| 参数 | 建议值 |
|------|--------|
| 格式 | MP4 (H.264 编码) |
| 分辨率 | 建议 720x1280 或 1080x1920（竖屏）|
| 时长 | 5-10秒（视频会循环播放直到接通）|
| 文件大小 | 建议不超过 10MB |

## 使用场景示例

### 场景1：号码已绑定彩铃

```
用户拨打 13800138000
↓
查询数据库：13800138000 绑定了 彩铃A
↓
播放 彩铃A 的视频
不播放拨号等待音
```

### 场景2：号码未绑定彩铃

```
用户拨打 13300133000
↓
查询数据库：13300133000 未绑定彩铃
↓
尝试随机播放数据库中的彩铃
↓
如果也没有 → 隐藏视频
播放拨号等待音
```

### 场景3：多个号码共用同一个彩铃

```
彩铃A 绑定了：13800138000、13900139000、13700137000
拨打这三个号码都会播放 彩铃A
```

## 注意事项

1. **号码唯一性**：一个号码只能绑定一个彩铃，重复绑定会覆盖之前的
2. **自动解绑**：删除彩铃时会自动解绑所有关联的号码
3. **异步查询**：号码绑定查询是异步的，不会阻塞UI
4. **回退机制**：如果号码没有绑定，会尝试其他方式播放彩铃
5. **没有彩铃时**：隐藏视频，播放拨号等待音，应用正常工作

## 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 号码绑定了彩铃但不播放 | 视频路径错误 | 检查数据库中彩铃的 videoUri 是否正确 |
| 绑定后播放了其他彩铃 | 数据库缓存 | 重新查询数据库或重启应用 |
| 解绑后仍播放彩铃 | 使用了随机播放 | 确认是否移除了所有彩铃 |
| 视频播放卡顿 | 视频分辨率过高 | 建议使用 720p 分辨率 |
| 批量导入失败 | 号码格式错误 | 确认号码是11位手机号 |
