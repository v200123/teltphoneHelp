# 彩铃视频使用说明

## 功能说明

应用现在支持在接通电话前播放彩铃视频。彩铃视频来源优先级如下：

1. **Room 数据库中用户上传的彩铃**（最高优先级）
   - 支持多个彩铃，随机选择一个播放
   - 通过彩铃管理页面上传

2. **res/raw 目录下的默认视频**
   - 作为后备方案

3. **assets 目录下的视频**
   - 第二后备方案

4. **外部存储的视频**
   - 最后的后备方案

## 优先级说明

```
Room数据库(随机选择) > res/raw > assets > 外部存储
```

## 如何添加彩铃视频

### 方式一：通过应用内上传（推荐）

通过应用内的「彩铃管理」功能上传视频：
1. 打开应用的彩铃管理页面
2. 点击添加/上传视频
3. 选择本地视频文件
4. 上传成功后，拨打电话时会自动随机选择一个彩铃播放

上传的彩铃会存储在 Room 数据库中，支持：
- 存储多个彩铃视频
- 每次拨打电话时随机选择一个播放
- 可以删除或切换默认彩铃

### 方式二：放到 res/raw 目录（开发者）

1. 准备一段视频文件（建议格式：MP4，时长：5-10秒）
2. 将视频文件命名为 `ringtone_video.mp4`
3. 放到 `app/src/main/res/raw/` 目录下

### 方式三：放到 assets 目录（开发者）

1. 在 `app/src/main/assets/` 目录下创建 `video` 文件夹
2. 将视频文件命名为 `ringtone_video.mp4`
3. 放到 `app/src/main/assets/video/` 目录下

### 方式四：动态下载到外部存储

将视频文件下载到应用的外部存储目录：
```
Android/data/com.u2tzjtne.telephonehelper/files/ringtone_video.mp4
```

## 视频格式建议

| 参数 | 建议值 |
|------|--------|
| 格式 | MP4 (H.264 编码) |
| 分辨率 | 建议 720x1280 或 1080x1920（竖屏）|
| 时长 | 5-10秒（视频会循环播放直到接通）|
| 文件大小 | 建议不超过 10MB |

## 技术实现

### 数据库表结构

彩铃视频信息存储在 `RingVideo` 表中：

```java
@Entity
public class RingVideo {
    @PrimaryKey(autoGenerate = true)
    public int id;
    
    public String videoName;    // 视频名称
    public String videoUri;     // 视频路径/URI
    public String mimeType;     // 视频类型
    public long fileSize;       // 文件大小
    public long duration;       // 视频时长
    public long createdAt;      // 创建时间
    public boolean isSelected;  // 是否为当前选中
}
```

### 播放逻辑

1. 拨号后 2 秒开始播放彩铃
2. 优先从 Room 数据库读取所有彩铃
3. 如果有多个彩铃，随机选择一个播放
4. 如果数据库没有彩铃，尝试默认视频位置
5. 电话接通后自动停止播放并隐藏视频

### 代码调用

```kotlin
// 在通话界面中自动处理
VideoPlayerHelper.getInstance().playRingtoneVideo(context, packageName)

// 接通后停止播放
VideoPlayerHelper.getInstance().stopPlaying()

// 页面销毁时释放资源
VideoPlayerHelper.getInstance().release()
```

## 注意事项

1. **视频文件是可选的**，没有视频时应用会正常工作，视频区域自动隐藏
2. **多个彩铃随机播放**：如果数据库中有多个彩铃，每次拨打电话会随机选择一个
3. **视频循环播放**：彩铃视频会循环播放直到电话接通
4. **自动清理**：电话接通后视频自动停止并隐藏
5. **资源释放**：页面销毁时记得调用 `release()` 释放资源

## 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 视频不播放 | 视频路径错误 | 检查数据库中存储的 videoUri 是否正确 |
| 视频播放卡顿 | 视频分辨率过高 | 建议使用 720p 分辨率 |
| 只有声音没有画面 | 视频编码不支持 | 使用 H.264 编码的 MP4 文件 |
| 接通后视频还在播放 | 未调用 stopPlaying | 检查是否调用了接通状态回调 |
